Symplissime AI Widget Test

Intégration basique
Configuration minimale avec paramètres par défaut
<script src="symplissime-widget.js"></script> <div class="symplissime-chat-widget"></div>
Intégration standard
Configuration recommandée avec workspace et endpoint
<script src="symplissime-widget.js"></script> <div class="symplissime-chat-widget" data-api-endpoint="symplissime-widget-api.php" data-workspace="support-windows"></div>
Intégration personnalisée
Configuration complète avec tous les paramètres
<script src="symplissime-widget.js"></script> <div class="symplissime-chat-widget" data-api-endpoint="symplissime-widget-api.php" data-workspace="support-windows" data-title="Assistant IA" data-subtitle="Support technique" data-placeholder="Votre question..."></div>
Intégration serveur distant
Widget pointant vers une API externe
<script src="symplissime-widget.js"></script> <div class="symplissime-chat-widget" data-api-endpoint="https://monsite.com/api/chat.php" data-workspace="mon-workspace" data-title="Mon Assistant"></div>
Attributs disponibles
Liste complète des paramètres de configuration
data-api-endpoint
URL de l'API backend (défaut: symplissime-widget-api.php)
data-workspace
Workspace Symplissime (défaut: support-windows)
data-title
Titre affiché dans le widget (défaut: Symplissime AI)
data-subtitle
Sous-titre du widget (défaut: Assistant technique en ligne)
data-placeholder
Texte de l'input (défaut: Posez votre question...)
Intégration multiple
Plusieurs widgets sur la même page
<script src="symplissime-widget.js"></script> <!-- Widget support technique --> <div class="symplissime-chat-widget" data-workspace="support-windows" data-title="Support IT"></div> <!-- Widget vente --> <div class="symplissime-chat-widget" data-workspace="commercial" data-title="Aide commerciale"></div>
Chargement asynchrone
Pour optimiser la vitesse de chargement de page
<script src="symplissime-widget.js" async></script> <div class="symplissime-chat-widget" data-workspace="support-windows"></div>
Test dynamique
Ajouter et supprimer un widget pour vérifier l'absence de fuites mémoire
